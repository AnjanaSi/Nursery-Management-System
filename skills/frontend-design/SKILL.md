# Frontend UI Skill (Nursery Application)

You are a world-class UI/UX designer. When creating components:

## Design Style

- Warm, friendly, modern.
- Soft rounded corners.
- Subtle shadows.
- Bright but calm colors.
- Consistent spacing
- Consistent typography (Bootstrap defaults OK, avoid too many fonts)
- Plenty of whitespace and clear layout
- Forms should be clean:
  - Use Bootstrap form controls
  - Inline validation messages
  - Disable submit button while submitting

## Layout

- The application must be fully responsive and usable on:
  - Desktop (≥1200px)
  - Tablet (768px–1199px)
  - Mobile (<768px)
- Use Bootstrap 5 responsive grid system (`container`, `row`, `col-*`).
- Use responsive navbar with collapse toggle for small screens.
- Avoid fixed widths unless necessary.
- Tables must be wrapped in `table-responsive`.
- Forms must stack vertically on small screens.
- Sidebar (if used) must collapse or convert to top navigation on mobile.
- Test layouts using browser responsive mode before finalizing components.

## UX Rules

- Always include loading states.
- Always include empty states.
- Forms must show validation feedback.
- Avoid dense tables on mobile.

## Images & Branding

- Use nursery logo from assets folder.
- Use image placeholders if real images are missing.
- Maintain correct image aspect ratios.
- Never stretch logos.

## Public Landing Page Experience

- One-page scrollable layout.
- Smooth scrolling to sections from navbar links.
- Clear call-to-action buttons (Login / Contact).
- Hero section should feel welcoming for parents.
